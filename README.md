## How to run locally üíª

Follow these steps to run My Portfolio in your local machine.

### Install NPM

```bash
npm install
```

### Run

Run the development server to view the app.

```bash
npm start
```

Now, you can navigate to http://localhost:3000/ to view the app.

### Build

```bash
npm run build
```
